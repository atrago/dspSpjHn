# 项目简介

## 核心功能
- 角色设计：普通用户、农户、管理员
- 前台功能模块：首页展示、全部商品页面、购物车模块、订单模块、个人中心
- 后台功能模块：数据大屏模块、用户管理、订单管理、评价管理、公告管理、资讯管理

## 适用场景
- 农产品电商平台
- 农产品销售企业
- 农业合作社
- 农产品供应链管理

## 技术特点
- 采用Springboot3、Vue3构建，提供高效的项目开发和运行环境
- 使用协同过滤推荐算法，优化商品推荐功能
- 数据大屏展示，直观展示销售数据和用户数据
- Echarts相关技术，丰富数据可视化效果
